PROJECTS

Hriday

An AR companion and meditation chamber

Hriday embarks upon the theme of mental health awareness by providing the user with the user with an AR companion (a pet dog to be specific) which acts as a confidant and performs manoeuvres on being instructed. The app is also provided with an AR meditation chamber to engage in an immersive meditative experience.

Holocron

AR treasure hunt

Holocron is a riveting AR-based game based on the theme of Star Wars. Made using Unity 3D and Vuforia SDK, the game offers an engrossing treasure hunt experience. The player has to scan a set of images and then solve the riddles based on wordplays augmenting on them. The player has to clear numerous checkpoints to complete the game.

DiveARsity

AR effect + game for video calls

Encapsulating the essence of diversity of India, DiveARsity is a group effect for video calls on Instagram and Messenger built using Meta's Spark AR, Multipeer API and Participant API. The effect involves face tracking and blink detection of the user to navigate a game object across the hurdles and multipeer interaction to enjoy the game with other fellows on a video call.

Book A Beam

Book donation website

Ever voraciously searched for a book in the library a night before the exam only to realize later that a senior/friend had the same one resting on their bookshelf? Book-a-Beam embarks on solving the problem by facilitating hassle free exchange of books among college students. Built on Django framework, it also eases the book donation process.
